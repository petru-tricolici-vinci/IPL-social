## Description
Ce projet est un exemple d’intégration continue (CI) avec GitHub Actions et TDD. Le pipeline exécute automatiquement les tests à chaque push sur la branche `main`.

## Explication du code
Ce projet simule une application sociale basique avec des tests unitaires écrits en premier (TDD). Le pipeline CI vérifie que tous les tests passent avant toute intégration.
